Des commentaires dans du CSS

Pour faire un commentaire, vous tapez /*, suivi de votre commentaire, puis */ pour terminer votre commentaire.
p { color:blue; /* Les paragraphes seront bleus*/ font-size:18px; /* La taille de 18 pixels me semble pas mal*/ }
