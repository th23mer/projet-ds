Les balises universelles

Appliquer une class (ou un id) à certains mots qui ne sont pas entourés par des balises.

Exemple:

modifier uniquement “grand père” dans le paragraphe suivant :
<p>Comme l’a dit grand père un certain 20 juillet 1969...</p>
<span> </span> : c'est une balise de type inline, c'est une balise qui se met à l'intérieur d'un paragraphe, comme <strong>, <em>, <q>.
<div> </div> : c'est une balise de type block. Comme <p>, <h1> etc...
Elle crée un nouveau "bloc" dans la page, et provoque donc obligatoirement un retour à la ligne.

Exemple:

<p>Comme l'a dit <span class="nom">Neil
Armstrong</span>un certain
20 juillet 1969...</p>
Code CSS :
.nom {
color:blue;
}
Donc pour définir un style applicable seulement à une partie de la page et non plus à une balise <HTML> précise, vous pouvez placer un attribut style dans une balise <DIV> (division) ou <SPAN> ( fusion).
Ces balises définissent une zone de texte donnée, donc tout ce qui se trouve entre la balise de début et la balise de fin adopte le style défini.
