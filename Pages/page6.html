Imbrications de balises

  • div, p:
    • Sélectionne tous les éléments
    et tous les éléments <p>.
  • div p
    • Sélectionne tous les éléments <p> dans les éléments <div>.
  • div> p
    • Sélectionne tous les éléments <p> dont le parent est un élément <div>.
  • div + p
    • Sélectionne tous les éléments <p> placés immédiatement après les éléments <div>.
  • div ~ p
    • Sélectionne tous les éléments <p> précédés d'un élément <div>
