Taille du texte

En pixels :

c'est une façon très précise d'indiquer la taille du texte.
p { font-size:14px; /* Paragraphes de 14 pixels */ }
h1 { font-size:22px; /* Titres de 22 pixels */ }

En pourcentage :

Si vous indiquez "100%", le texte aura une taille "normale". Si vous mettez "130%", le texte aura une taille correspondant à 130% de la taille normale.

En donnant une valeur relative :

  • xx-small : minuscule
  • x-small : très petit
  • small : petit
  • medium : moyen
  • large : grand
  • x-large : très grand
  • xx-large : plus grand que le x-large

En em :

c’est une autre façon d’indiquer de manière relative la taille du texte.

On doit indiquer la taille du texte par rapport à la taille normale de la police. "1em" signifie "Taille normale".

.petit_em { font-size:0.7em; }

.grand_em { font-size:1.3em; }
