Introduction to CSS

Objectifs du Cours

À la fin de ce cours, vous serez capable de :
  • Maîtriser les bases et l'utilisation avancée du CSS pour styliser vos pages web
  • Créer des mises en page réactives et adaptatives
  • Utiliser les animations et transitions CSS pour dynamiser votre site
  • Exploiter des outils modernes de design pour une gestion de style plus efficace et plus rapide

Définition

  • CSS : c’est l’abréviation de Cascading Style Sheets ("Feuille de style en cascade").
  • C'est dans la feuille de style que l'on va déclarer toute la mise en forme des pages : la couleur de fond, les polices de caractère, leurs couleurs, etc.
  • HTML pour écrire le contenu de nos pages web.
  • CSS pour présenter ce contenu.
  • Ces 2 langages sont donc complémentaires.
